# Changelog

## 0.3.0

- Added `crumb pack` for deterministic context pack assembly with token budgets, git/repo signals, and mode-aware output shaping
- Added `crumb bridge mempalace export|import` as the first adapter-based bridge surface
- Added `crumb lint` for secrets, header hygiene, raw-log warnings, and budget checks
- Added extension helpers plus documented optional headers and namespaced metadata
- Added golden fixtures and broader Python/Node validator coverage
- Repositioned the docs around CRUMB as an open context interchange standard

## 0.2.0

- Added MeTalk compression and PyPI packaging for the CLI + SDK
